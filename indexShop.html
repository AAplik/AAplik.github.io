Shopping List App

Displays a shopping list. Can input information about price and category.

  • Portfolio

How it works

The app opens to a load screen. Then, it displays an empty list. When the user pushes the envelope button, an input screen opens with options to input the name, price, category, description, and whether or not it is purchased. For a few seconds after the item is created, the user can hit undo to get rid of the recently created item. The user can delete, edit, or display the details of the item. The "X" button will delete all entries.
