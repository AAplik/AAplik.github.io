How it works

This website was created with Vue and a Firebase database. On the home page, the user can select a dificulty level which changes how much the pictures are manipulated. Then, they can begin the game. There is a picture that is normal, pixelated, or inverted and pixelated, depending on the level and four possible answer buttons. The timer counts down and the longer the user takes to answer, the less points they get. The correct answer turns green, and the wrong answers turn red. Once the user has used up all three lives, there is an end screen where the user can input their name and a leaderboard with high scores appears.
