Watch List App

Shows a list of movies with a picture of the poster and the IMDB rating.

  • Portfolio

How it works

On the main page, the user can hit the envelope button which will take them to the search page. The user can input text into the search bar and hit the search button. All the movies that match the text will display with the poster, the title, the year, the rating, and the description. If the user hits the "Save to List" button, they will automatically be taken back to the list page with the new movie included. For a few seconds, the user can select undo on the popup to delete the movie they just added. The user can scroll up and down on the list and hit delete on any of the movies.
